# Smiling School - JavaScript

Build a web app in JavaScript for Holberton School. This project implements
fully functional web pages (homepage, pricing, courses) using Bootstrap,
jQuery and dynamic data loading from a REST API. Static content is replaced
by Ajax requests with loaders, working carousels, and dynamic filtering.

## Technologies
- HTML5 / CSS3 / Bootstrap 4.4.1
- JavaScript / jQuery 3.4.1
- REST API (smileschool-api.hbtn.info)
